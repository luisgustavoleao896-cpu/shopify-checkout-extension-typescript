# Shopify Checkout UI Extension – TypeScript

Projeto demonstrando:

- Checkout UI Extension com React
- TypeScript
- Shopify Functions
- Estrutura modular
- Arquitetura compatível com Shopify Plus

## Funcionalidades

- Upsell exibido no checkout
- Banner promocional
- Function para desconto automático
- Estrutura organizada em extensions e functions

## Stack

- React
- TypeScript
- Shopify CLI (estrutura simulada)
- GraphQL-ready
